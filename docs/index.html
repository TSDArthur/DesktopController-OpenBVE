RAGDOLL CATS PLATFORM

基于久经考验的 Arduino 开源硬件与 OpenBVE 开源列车模拟器，一个开源、稳定、强大、可扩展的列车驾驶模拟硬件设备开发平台。

进入代码仓库

平台特性

我们将成熟的 C++ 开发经验完全运用到基于 Arduino Processing 语言的硬件开发中， 并不断提升效率，优化性能。至此，久经考验的平台拥有了极为现代化的灵魂。

可扩展性

从现在开始，自定义一台低成本、高可靠性的虚拟列车驾驶台只需要三步。所有的一切完全开源，选择您的OpenBVE车辆与线路 -> 修改功率手柄与制动限位值 -> 接入开关与线路，只需要三步，即可搭建虚拟列车驾驶台硬件。

高可靠性

异步串行接口数据传输，Ping-Pong式数据传输，两种数据覆写模式，数据帧起始校验，软硬件联合消抖。共同构成了高可靠性的列车驾驶模拟体验。

为Geek而生

我们提供了一系列适合从 Arduino 新人到工程师的特性。整体运行架构由有限状态自动机、等候队列与定时器中断构成。支持三种设备唤起方式，可无限添加操作设备并按规范为其编写代码。我们甚至在上位机中提供了主定时器，以供完成高级操作。
